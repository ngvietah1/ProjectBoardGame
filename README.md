# ProjectBoardGame

**ProjectBoardGame** — это настольное приложение на WPF для ведения каталога и учёта настольных игр.

---

## Описание

Приложение позволяет:
* Просматривать подробный список настольных игр различных типов (EuroGame, PartyGame, CardGame) и их характеристики (количество игроков, возрастные ограничения, описание).
* Выводить детальную информацию о выбранной игре в отдельном окне.
* Сохранять и загружать каталог игр (поддерживается сериализация/десериализация данных).

---

## Основные возможности

* Отображение полного каталога игр в виде таблицы с возможностью быстрого поиска и фильтрации.
* Просмотр подробных характеристик, возрастных ограничений и описания каждой настольной игры в отдельном удобном окне.
* Добавить, удалить игру.
* Фильтрация по типу, количеству игроков, возрастному ограничению.
* Анализировать свойства игр.

---

## Технологии

* **C# 12 / .NET 10.0 (Windows)**
* **WPF** (Windows Presentation Foundation)
* Сериализация данных в формате Json.

---

## Структура проекта

Решение разделено на два основных проекта согласно архитектурным шаблонам:

1. **`Model` (Класс-библиотека):**
   * Содержит бизнес-логику и основные модели в папке `Core` (`BoardGame.cs`, `PartyGame.cs` etc.).
2. **`UIBoardGame` (WPF-приложение):**
   * Пользовательский интерфейс приложения.
   * `MainWindow.xaml` — главное окно со списком игр, фильтрами.
   * `NewWindow.xaml` — модальное окно для детального просмотра характеристик конкретной игры.
   * `AnalizeWindow.xaml` - окно для таблицы игр.
   * `AddWindow.xaml` - окно для добавления игры.
   * `Analyze.xaml` - окно для анализа.

## Как запустить проект

1. **Клонируйте репозиторий** на свой локальный компьютер.
2. Откройте файл решения нового формата **`ProjectBoardGame.slnx`** в среде разработки **Visual Studio 2022** (версии 17.10 или выше, поддерживающей файлы `.slnx`).
3. Убедитесь, что целевая платформа — **`.NET 10.0-windows`**.
4. Проект `UIBoardGame` автоматически настроен как главный (**Startup Project**). Если этого не произошло, нажмите правой кнопкой мыши на `UIBoardGame` в Solution Explorer и выберите *Set as Startup Project*.
5. Нажмите **F5** (или *Start without debugging*) для запуска приложения.
